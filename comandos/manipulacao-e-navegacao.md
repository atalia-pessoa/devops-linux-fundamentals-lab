### 1: Listar todos os diretorios

```Bash
ls
```

### 2: Listar arquivos ocultos

```Bash
ls -a
```
### 3: Criando diretorios

```Bash
makdir nome_pasta
```

### 4: Criando arquivos

```Bash
touch nome_arquivo.txt
```

### 5: Navegar em diretorio

```Bash
cd nome_pasta
```

### 6: Historico de todos o comandos utilizados

```Bash
history
```

### 7: Ler e exibir arquivos

```Bash
cat arquivo.txt
```
